Info about Mediatout

Mediatout is a library management platform.

Mediatout Admin is a Windows application designed to manage the Mediatout website and mobile app (a monitoring app). This project was created using .NET in C#, a language I learned during my Bac+2 training. Additionally, I used the structural language Guna to enhance the UI/UX.

Our teachers decided to organize us into groups to expedite the execution of our tasks and to ensure proper distribution. During this production, I was responsible for both the mobile and desktop admin parts.

This administration application offers various functionalities, including user management, visualization of library resources, stats visualization, and resource management.
